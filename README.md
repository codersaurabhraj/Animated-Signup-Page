# 🔐 Animated Signup Page

A modern and responsive Signup Page UI built using HTML and CSS.  
This project features smooth animations, glassmorphism design, and a clean user interface for user registration.

---

## 🚀 Features
- ✨ Smooth animations  
- 🧊 Glassmorphism UI (blur effect using backdrop-filter)  
- 📱 Fully responsive design  
- 🧾 Clean and minimal signup form  
- 🔒 Password & confirm password fields  
- 🎯 Gender selection (radio buttons)  
- 🌍 Country dropdown selection  

---

## 🛠 Tech Stack
- HTML5  
- CSS3
